# Code of Conduct

Gomoku Mini follows a simple standard for community participation: be respectful, constructive, and welcoming.

## Expected Behavior

- Use friendly and inclusive language
- Respect differing opinions and experience levels
- Give feedback that helps the project improve
- Assume good intent where reasonable

## Unacceptable Behavior

- Harassment or personal attacks
- Discriminatory language
- Public or private intimidation
- Repeated disruption of project discussions

## Reporting

If you see behavior that makes the project less welcoming, open a private discussion with a maintainer or use the contact method listed by the repository owner.
